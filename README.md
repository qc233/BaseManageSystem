## 登陆注册页面



页面整体节能环保，实现 **一框两用**。



### 请求处理

项目为**服务端客户端**混合，客户端直接调用相关函数向服务端请求数据。

由**Controller**直接接收请求 -> 转交**Service**实现类数据处理 -> **MyBatisPlus**实现**DAO**完成数据库交互 -> **Controller**返回结果。
